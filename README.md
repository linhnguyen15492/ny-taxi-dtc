# New York City Taxi Fare Prediction

This is final project for data engineering zoomcamp from **datatalks.club**. The goal of the project is to predict the fare amount for a taxi ride
in New York City based on various features such as pickup and dropoff locations, time of day, and passenger count.

The project involves data preprocessing, feature engineering, and model training using machine learning algorithms. The
dataset used for this project is the New York City Taxi Fare Prediction dataset, which contains information about taxi
rides in New York City, including pickup and dropoff locations, timestamps, and fare amounts.

## Reproduction steps

1. Clone the repository: `git clone https://github.com/yourusername/ny-taxi-dtc.git`
2. Navigate to the project directory: `cd ny-taxi-dtc`
3. Install the required dependencies: `uv add -r requirements.txt`
4. run script to create credentials for GCP: `mkdir -p .credentials`
5. copy sample credentials file to .credentials directory: `cp gcs.example.json .credentials/gcs.json` and fill in the required fields
